##背景
随着无线搜索业务逻辑不断增加，项目越来越多，且需求趋向于运营化，这对目前的前端程序运行框架提出更高的要求，我们需要赋予这个框架更多灵活更多扩展能力。 

为了达到这个目标，需要将前端框架抽象化平台化，摆脱目前的项目开发模式，更高效更灵活的满足业务需求。


首先要做的事，就是将目前的PHP环境搭建整理，并使其自动化部署。

目前我们使用的PHP包括PHP本身和PHP扩展，而后者由于其编译困难给环境部署升级造成了障碍。

为此，我们开发了PHP扩展自动编译脚本，将扩展的编译、安装、配置自动化，一键完成。
